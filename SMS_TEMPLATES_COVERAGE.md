## Implementation Details

### Files Modified
1. **lib/alert-templates.ts**
   - Added `BANK_SMS_TEMPLATES` object with templates for all 48 institutions
   - Added `getBankTemplates()` function to retrieve templates by bank name
   - Updated alert generator functions to accept bankName parameter
   - Added `getAllSMSTemplates()` function to retrieve all templates across all banks

2. **lib/sms-service.ts**
   - Updated `getDefaultTemplates()` to generate templates for all banks
   - Added `getTemplatesForBank()` method to get all templates for a specific bank
   - Added `getTemplatesByType()` method to filter templates by alert type
   - Added `getBanksWithTemplates()` method to retrieve all banks with template coverage

### Template Variables
Each SMS template includes the following standard variables:
- `{amount}` - Transaction amount
- `{recipient}` - Recipient name/account
- `{sender}` - Sender name/account
- `{balance}` - Account balance
- `{reference}` - Transaction reference number

### Usage Examples

#### Get templates for a specific bank:
```typescript
import { getBankTemplates } from "@/lib/alert-templates"

const ecoTemplates = getBankTemplates("Ecobank Nigeria")
// Returns: { debit: "...", credit: "...", balance: "...", lowBalance: "..." }
```

#### Generate a debit alert for a specific bank:
```typescript
import { generateDebitAlert } from "@/lib/alert-templates"

const alert = generateDebitAlert(
  50000,           // amount
  "John Doe",      // recipient
  150000,          // balance
  "REF123456",     // reference
  "Zenith Bank"    // bank name
)
```

#### Get all templates from SMS service:
```typescript
import { SMSService } from "@/lib/sms-service"

const allTemplates = SMSService.getDefaultTemplates()
// Returns array of 192 templates (4 per bank × 48 banks)
```

#### Get templates for a specific bank:
```typescript
const bankTemplates = SMSService.getTemplatesForBank("Access Bank")
// Returns array of 4 templates (debit, credit, balance, low balance)
```
