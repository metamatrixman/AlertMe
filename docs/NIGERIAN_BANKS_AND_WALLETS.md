## Implementation Guide

### Using Bank Selection in Forms

#### 1. Simple Bank Selection

```typescript
import { NIGERIAN_BANKS } from "@/lib/banks-data"
import { formatBankOptionsFlat } from "@/lib/bank-selection-utils"

// Get all banks as flat options for Select component
const bankOptions = formatBankOptionsFlat()

// Filter specific types
const banksOnly = formatBankOptionsFlat("banks")
const walletsOnly = formatBankOptionsFlat("wallets")
const microfinanceOnly = formatBankOptionsFlat("microfinance")
```

#### 2. Grouped Bank Selection

```typescript
import { formatBankOptionsForSelect } from "@/lib/bank-selection-utils"

// Get banks grouped by category for better UX
const groupedOptions = formatBankOptionsForSelect()
```

#### 3. Form-Specific Configuration

```typescript
import { getBankConfigForFormType } from "@/lib/bank-selection-utils"

// Get configuration for specific form type
const config = getBankConfigForFormType("domesticTransfer")
// Returns: { label, description, options, placeholder, required }
```

#### 4. Validation and Lookup

```typescript
import { 
  validateBankSelection,
  getBankName,
  getBankCode,
  isWallet,
  requiresPhoneNumber 
} from "@/lib/bank-selection-utils"

// Validate bank code
const validation = validateBankSelection("044") // Access Bank
if (validation.valid) {
  console.log(validation.bank?.name)
}

// Get bank info
const name = getBankName("044") // Returns "Access Bank"
const code = getBankCode("Access Bank") // Returns "044"

// Check if it's a wallet
const isPhoneWallet = isWallet("999992") // Returns true for Opay
const needsPhone = requiresPhoneNumber("Opay") // Returns true
```

### Data Consistency

All form components that require bank selection should use:

1. **Import the banks data:**
   ```typescript
   import { NIGERIAN_BANKS } from "@/lib/banks-data"
   ```

2. **Use utility functions for consistency:**
   ```typescript
   import { formatBankOptionsFlat, getBankConfigForFormType } from "@/lib/bank-selection-utils"
   ```

3. **Never hardcode bank lists** - Always reference the centralized data source

### Updating Bank Information

To add new banks or update existing ones:

1. Edit `/lib/banks-data.ts`
2. Add/update the bank entry in the `NIGERIAN_BANKS` array
3. Ensure proper type classification (bank, wallet, microfinance, international)
4. All components automatically reflect the changes
